  1. What are different ways to style your form?

    We can use many ways, if we only use the most basic simple one without CSS, just use br in our HTML and let it layout each line.
    but, we do not want to use that, instead of that we can make it each box with the div tag so it will easier to lay out them,
    then, use the flex to arrange their position, also we can use pseudoclass focus to make it accesible when the user click it.
    There are so many different way to style our form, flex; grid; float. Use the easier one with the right condition that your pages have.

  2. Why does the professor like to use flex?

    Because flex really easy and helpful to arrange the layouting position. Not only the professor but I think most of web dev use flex a lot.
    Some reason because flex are able to modify to row and column, which is right, left, top, bottom. It is so easy to arrange our element HTML.
