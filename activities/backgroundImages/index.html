  1. When should you use a background image, and when should you use the IMG tag?

    We use background image when we want to make a picture as a background of our web, we can choose to use that image as a background of our body tag (whole page) or header, footer, main like that, we use backgroung image property in our CSS then use "url()" to call and put the path of the image
    In other side, we use Image tag in HTML when we want to put an image in our web page, the image not as background but only image that we want to put in our web page, and it's as a tag in HTML, different with background image that as property in CSS.
