  5. What goes in Main?

    The content. So, that is why it called Main, so it contains the content for the webpages, just let's say from the navigation links that we have each navigation has different content, so in this section (main) it will contain the content for each major nav

  6. What goes in the Footer?

  7. HTML Validation

    A validated website is easier to read by browsers such as Chrome, Mozilla, so make sure our HTML is validated, go to validator.w3.org and paste our HTML there and if there is an error it will appear, fix it and if it is safe and green your website is good.
