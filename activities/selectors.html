Answer the following questions:

  1. List out the different ways to select an item.

    A. Tag Name: We use the tag name, example: body, p, h1, etc.
    B. ID: Create id="name", then in CSS we use "#" like "#name" to select that ID. NOTE: Can't reuse
    C. Class: Create class="name", then in CSS we use "." like ".name" to select that class. NOTE: Can have more than 1 class
    D. Descendent: We can select the specific ID like that by select the tag name first then following the ID name. Example: main(tag name) .first(ID name).

  2. What is the difference between an ID and a Class?

    We can only use ID for 1 name each in our HTML file, so it can not reuse, so specific. In other hand, class is can be use for more than 1 class, so we can apply the same name class for many element in our HTML file.

  3. What is the difference between a SPAN tag and a DIV tag?

    Div: It creates Inline box (Create a line break), Span: Creates Block Box (Create only as big as the content) so it does not takes any line break in the screen.

  4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

    So, we use these tags only for syntax, this tag does not outputing anything, but only creates some content that we can use to inserting some value. The differences between Div, Header, Main and Footer is the meaning or purposes.
    Div: Using to make some content but not specifics, for example: if we want to create boxes that contains photo, we can use div only for the container.
    Header: Using to keep our header content such as Nav and Title header.
    Main: Using to keep our main content in our web.
    Footer: Using to keep our copyright things.

  5. What are different ways to select a color?

    We can use only the name of the color, using RGBA, using # following the code of the color, etc. Up to us.
