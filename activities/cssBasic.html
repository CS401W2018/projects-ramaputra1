  1. Explain the difference between inline, embedded, and external CSS.

    Inline: We use this way by adding our css style inside our tag in HTML, like inside our tag p we have style with blue color in our text. And it is not really effective, so Do Not use this.

    Embedded: We use this way by adding tag style in our HTML head (below the title), and we styling our website by using CSS attribute there. But, just do not use this way, because it will make our HTML file not clear (not effective).

    External CSS: We use this way by creating another file, and name this by fileName.css and then just link your fileName.css with your HTML file by using tag link in HTML head.

  2. Explain why you should only use external CSS.

    It's better to use External CSS because it is more Effective, Clean and Clear., instead of using the other two ways (Inline and Embedded), using external CSS way is will help us to read clear and looks more clean for the validation and it will impact as well for the browser I guess and when we doin with JS, so Use external CSS.
