Engagement Questions

  1. Why do we need a viewport meta tag?

    We need Meta tag viewport because it helps our website to be responsive, it will adjusting the website to different other devices like phone, tablet, desktop.

  2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

    In CSS, the last thing in the stylesheet will take precedence. So it depends which property in that tag that be in the last, if the flex-direction: row in the last, CSS will use that.

  3. How do you change the CSS for a web page displayed on a phone?

    We change that using media queries, by type @media then say for the screen and set the width and each break point, then we can apply what we want in that range of width. Let say in the phone width we want the nav to be in the hamburgermenu or something like that.
