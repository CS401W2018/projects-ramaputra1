Engagement

  1. What is the difference between a block and an inline tag? Give examples of each one.

    The differences is that block will have a full of the screen as widht so it will make a new line,
    and inline will only take as big as content for the width so it will required a new line.
    Example:
    Inline: span, a, img.
    Block: p, h1, h3

  2. What is normal flow in HTML?

    Normal flow is the default flow in HTML CSS, it allows each block element to have new line in ordered. So, each block tag wants to take the entire width of the screen. Then, if we have a tag inside of a block tag it will also take up the width of the previous block tag (parents). Then, next block tag will take new line. That is normal flow.

  3. What does float do? Why do you need to clear the float?

    Float allowed us to make the img that we have or element block that we have to be on the right or left side, then if we have text there we can make our text to be in the next of our image, not in the below of the picture.
    We want to clear the float so it will not be collaps, because the flow being crash, so we need to make empty div and use that div with clear property in CSS, to be clear right, left or both.

  4. What do absolute, fixed, and relative do?

    All these property work to positioning the content that we have. It works like move that content to another dimension and then it will allow us to move the content everywhere that we want using top, bottom, left, right with measurement there.
    The difference is:
    absolute: move to another dimension but the place or space before is not reserved.
    relative: opposite of the absolute, the space before reserved.
    fixed: same but it will fixed there, not move anywhere.

  5. How can you make nice navigation buttons for mobile devices?

    First, if we use ul just list-style-type = none, then make it block so we can give padding to have space there, and style it with border and bg color, then use hover as pseudoclass to transform your nav to be different when it on hover. if you want to make it horizontally use in your ul float, but dont forget to clear it.
